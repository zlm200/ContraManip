# ContraManip

This is the official PyTorch implementation for the paper: **"ContraManip: Visual Contrastive Reinforcement Learning with Dual Data Augmentations for Robust Robotic Manipulation"**.

## Overview
ContraManip is a visual reinforcement learning framework that utilizes:
- **Dual Data Augmentation:** To generate diverse views for robust feature learning.
- **Contrastive Learning:** To enforce feature consistency and isolate interference factors.
- **End-to-End Training:** Integrating feature extraction (ResNet18) and policy learning.
